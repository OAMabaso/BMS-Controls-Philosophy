# BMS Controls Philosophy

Master template and structure for Building Management System (BMS) Control Philosophy & Operating Manual documents.

## Purpose

This repository contains the universal template for authoring professional BMS control philosophy documents for any site — chiller plants, AHUs, FCUs, ventilation systems, hot water plants, and common building services.

## Files

- `BMS_Control_Philosophy_Template.md` — The complete master template with all sections, guidance, data capture tables, and compliance checklists.
